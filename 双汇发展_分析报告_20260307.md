**投资评级**: **HOLD（持有）**

### 5.3 终端价值计算

**永续增长法:**
```
终端价值 = FCF2030 × (1 + g) / (WACC - g)
        = 50.88 × 1.02 / (0.105 - 0.02)
        = 610.56 亿元
```

**退出倍数法:**
```
终端价值 = EBITDA2030 × 退出倍数
        = 55.38 / 0.08 × 7.0
        = 484.58 亿元
```

## 十一、结论

双汇发展是一家**防御性价值投资标的**，适合追求稳定现金流和分红的长期投资者。

**核心优势**:
- 行业龙头地位稳固
- 高分红收益率（5.14%）
- 财务结构稳健

**核心劣势**:
- 增长乏力
- 估值不低
- 周期性风险

**投资策略建议**:
1. **现有持有者**: 可继续持有，享受稳定分红
2. **潜在买入者**: 等待股价回落至22-24元区间（PE 15-16倍）
3. **风险控制**: 密切关注猪价走势和消费复苏情况
