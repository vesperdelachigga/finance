## 一、执行摘要

中国中免（601888.SH）是中国最大的免税商品运营商，当前股价 **76.62元**，总市值 **1585亿元**。

**重大事件**: 2026年2月24日，中国中免股价跌停，主要原因是公司失去了上海机场部分免税经营权。

**核心观点:**
1. 🔴 **重大利空**: 失去上海机场部分经营权，这是公司最重要的收入来源之一
2. 🔴 **业绩大幅下滑**: 2024年净利润下降36.44%，2025年前三季度继续下降22.13%
3. 🔴 **估值严重高估**: PE 37倍，远高于业绩增速，存在大幅回调风险
4. ⚠️ **分红不可持续**: 分红率143%，超过净利润，难以长期维持

**投资评级**: **SELL（卖出）**

### 5.3 估值与业绩增速匹配度分析

```
PEG = PE / 净利润增长率

当前PEG = 37 / (-22%) = 负值（业绩负增长）

PEG分析结论:
- PEG < 1: 低估
- PEG = 1: 合理
- PEG > 1: 高估
- PEG < 0: 业绩负增长，估值极高

中国中免当前处于业绩负增长阶段，但估值高达37倍，属于典型的"业绩下滑、估值高企"的危险组合。
```

## 十二、结论

中国中免（601888.SH）是一家**高估值的困境企业**，存在巨大的投资风险。

**核心问题:**
1. 🔴 **失去上海机场部分经营权** - 核心收入来源受损

**估值结论:**
- DCF合理价值: 20-25元
- PE 25倍合理价值: ~49元

- **下行空间: -35% 至 -75%**

**投资建议:**
- **现有持有者**: 建议减仓或清仓
- **潜在买入者**: 坚决不建议买入
- **观望投资者**: 等待股价回落至30元以下再考虑

**风险提示:**
1. 公司可能继续失去机场经营权
2. 业绩可能继续下滑
